# Dumbbell Tracker

Track the current weight you use per exercise on adjustable dumbbells (5–52.5 lbs, 2.5 lb increments). Data persists in localStorage. Designed as a mobile-friendly single HTML file for GitHub Pages, bookmarked on iPhone.

## Stack

Vite + vanilla JS + `vite-plugin-singlefile`

## Usage

```
npm run dev       # local dev server
npm run build     # outputs dist/index.html (single file, deploy this)
```
